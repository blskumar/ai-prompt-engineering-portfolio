# ai-prompt-engineering-portfolio

A portfolio of structured system prompts for transforming raw requirements into clear Markdown features.

## Purpose

This repository contains reusable system prompts designed to convert unstructured inputs such as notes, emails, stakeholder requests, and rough feature ideas into clean, structured Markdown output.

## Contents

### Prompt 1: Structured Feature Specification
Converts raw requirements into a well-organized feature specification with sections such as summary, business goal, functional requirements, inputs, outputs, constraints, and open questions.

### Prompt 2: Backlog-Ready Requirement Formatter
Transforms messy stakeholder notes into backlog-ready Markdown using objective, user story, acceptance criteria, scenarios, assumptions, and clarifications needed.

### Prompt 3: Implementation-Ready Feature Brief
Converts unstructured notes, chats, and emails into implementation-ready feature briefs with context, actors, triggers, main behavior, validation rules, exceptions, dependencies, and definition of done.
